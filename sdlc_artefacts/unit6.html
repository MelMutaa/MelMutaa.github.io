Learning Overview

Learning Outcomes:
  • Utilise linters to ensure high-quality Python code development.
  • Understand the various contributions from linters, appreciating how they achieve similar objectives while recognising their applicability in different contexts.
  • Refine Python code to be error-free, ensuring consistency in its design and structure.

I am be able to:
  • Confidently use linters to support the development of high-quality Python code.
  • Evaluate and compare linter outputs to understand their relevance and suitability for different development and testing environments.
  • Write Python code that is free from errors, maintains design consistency, and adheres to recognised Python standards for quality.
