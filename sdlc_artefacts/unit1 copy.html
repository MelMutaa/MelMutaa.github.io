Introduction to Secure Software Development

Learning Overview

Learning Outcomes:
  • Understand the basics of web development.
  • Create responsive layouts using HTML and CSS.
  • Implement interactive features using JavaScript.
  • Explore Waterfall and Agile approaches for secure software development.
  • Understand the role of Unified Modeling Language (UML) in supporting software development.
  • Familiarise with industry standards for secure software development.
  • Appreciate the importance of a risk-aware culture within organisations.

On completion of this unit, I am now able to:
  • Identify academic literature on secure software development using Agile processes.
  • Break down activities into tasks represented using UML flowcharts.
  • Explain the significance of fostering a risk-aware culture in addition to creating secure software.
